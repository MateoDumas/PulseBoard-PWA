# Public Assets

Coloca aquí los siguientes archivos para completar la PWA:

## Iconos Requeridos

- `pwa-192x192.png` - Icono 192x192px
- `pwa-512x512.png` - Icono 512x512px
- `favicon.ico` - Favicon (opcional)
- `apple-touch-icon.png` - Icono para iOS (opcional)

## Generar Iconos

Puedes usar herramientas online como:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

O crear manualmente con cualquier editor de imágenes.

## Estructura de Carpetas

```
public/
 ├── manifest.json (ya existe)
 ├── pwa-192x192.png (crear)
 ├── pwa-512x512.png (crear)
 └── favicon.ico (opcional)
```
